# Prueba ITTI

Aplicacion Mobile con Clean Architecture desarrollada dada la premisa de la prueba tecnica enviada.

## Features
- Login con validaciones.
- Consultas http.
- Dark/Light mode.
- Seleccion the temas.
- Onboarding View.
- NavigationDrawer.
- Clean Architecture.

## Requisitos

Version de Flutter: 3.7.12
